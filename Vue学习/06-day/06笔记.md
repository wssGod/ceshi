## 1.Promise（ES6语法）
* 做什么的？ Promise是异步编程的一种解决方案
  ```javascript
  // 参数 -> 函数(resolve, reject)
  // resolve,reject 本身又是函数
  new Promise((resolve, reject) => {
    // 异步操作
    setTimeout(() => {
      // 调用resolve后会走then函数(成功调用)
      resolve()
    }, 1000)
  }).then(() => {
    console.log('Hello,word')
    return new Promise((resolve, reject) => {
      // 异步操作
      setTimeout(() => {
        // 调用resolve后会走then函数
        // resolve()
        // 错误的时候调用catch方法
        reject('error')
      }, 1000)
    })
  }).then(() => {
    console.log('Hello,vue')
  }).catch((data) => {
    console.log(data)
  })
  ```
